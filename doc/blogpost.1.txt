===========
Stuart Rackham <srackham@methods.co.nz>


NAME
----
blogpost - Wordpress weblog client for AsciiDoc


SYNOPSIS
--------
*blogpost* ['OPTIONS'] 'COMMAND' ['POST_ID'] ['BLOG_FILE']


DESCRIPTION
-----------
Wordpress weblog client for AsciiDoc. COMMAND can be one of: create,
delete, list, update. POST_ID is blog post ID number (or . for most
recent post).  BLOG_FILE is AsciiDoc text file.

OPTIONS
-------
*--version*::
  Show program's version number and exit.

*-h, --help*::
  Show this help message and exit.

*-f, --conf-file*='CONF_FILE'::
  Configuration file.

*-p, --publish*::
  Set blog post status to published.

*--html*::
  BLOG_FILE is an HTML file not an AsciiDoc file.

*-t, --title*='TITLE'::
  Blog post title.

*-n, --dry-run*::
  Show what would have been done.

*-v, --verbose*::
  Increase verbosity.
